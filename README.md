# Ubuntu-Shell-Script

### 로그인셀 확인
$ echo $SHELL

### 파일 형식 확인
$ file script


### 문법 확인
$ sh -n script.sh

### 파일에 실행 권한 부여
$ chmod +x script.sh

###  스크립트 실행하기
$ ./script.sh
